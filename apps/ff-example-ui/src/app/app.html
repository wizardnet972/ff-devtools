Feature Flags Demo App

This is a demo page to test the Feature Flags Chrome Extension

Instructions

  1. Open the Feature Flags extension from your Chrome toolbar
  2. The extension should automatically detect the feature flags from this page
  3. If not, click the "Refresh" button in the extension
  4. Toggle the feature flags in the extension to see them update here

Console Logs

Open your browser console to see communication logs between the page and extension.
