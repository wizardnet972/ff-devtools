# Privacy Policy for Feature Flag Manager

**Last Updated:** December 11, 2024

## Overview

Feature Flag Manager is a Chrome extension that helps developers manage feature flags directly from Chrome DevTools. This privacy policy explains how we handle your data.

## Data Collection

**We do not collect, store, or transmit any personal data or information.**

## Local Data Storage

The extension uses the following local storage mechanisms:

1. **Browser localStorage**: Used only to store your theme preference (light/dark/system mode). This data:
   - Stays entirely on your device
   - Is never transmitted to any server
   - Can be cleared by uninstalling the extension or clearing browser data

2. **In-Memory Data**: Feature flags data is stored temporarily in memory while the extension is active. This data:
   - Is never persisted to disk
   - Is cleared when the extension is closed
   - Remains within your browser

## Data Transmission

**The extension does not transmit any data to external servers.** All communication is:
- Local to your browser
- Between the extension and the web pages you visit
- Used solely for displaying and managing feature flags in DevTools

## Permissions

The extension requires the following permission:

- **"tabs" permission**: Used to communicate with web pages to retrieve and manage feature flags. This permission is essential for the extension's core functionality and does not involve any data collection or transmission.

## Third-Party Services

This extension does not use any third-party analytics, tracking, or data collection services.
